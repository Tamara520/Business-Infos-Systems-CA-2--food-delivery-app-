 Food Delivery Mobile App System
 Project Overview
This project is developed as part of the Business Information Systems module at CCT College Dublin. The aim of this project is to design and analyze an information system for a food delivery mobile application.

The system focuses on improving efficiency, enhancing user experience, and ensuring secure and reliable services.


 Objectives
- Improve food delivery speed and tracking
- Provide secure payment and data protection
- Enhance customer experience
- Support business decision-making



 Technologies Used
- HTML
- CSS
- JavaScript
- Cloud Services (AWS / Firebase – concept)



 Features
- User registration and login
- Menu browsing
- Order placement
- Real-time order tracking
- Secure payment system


 System Qualities
 Security
- HTTPS Encryption
- Secure login system

 Reliability
- Cloud hosting
- Backup systems

Scalability
- Handles multiple users
- Auto-scaling capability

Legal & Ethical Considerations
- GDPR compliance for user data protection
- Secure handling of payment information
- User privacy protection
- Fair service practices

 Future Improvements

- Advanced security features
